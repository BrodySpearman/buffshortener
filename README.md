# B.U.F.F. (The Big URL Functional Formatter)

Trying to post long URLs that often break the character limits on social media apps like Twitter? Replace long and ugly URLs with short and elegant ones.

# Using the shortener

B.U.F.F. made to be as simple and straightforward as possible, available for everyone to use. All you have to do is add a link and it will be instantly reformatted, then added to a list. Links made without an account are only active for 24 hours.

No sign ups are required, however, creating an account allows you to:  
- Override the default 24 hour link lifespan (Up to 2 weeks).
- Increase your cap on the amount of URLs you can have at one time (10 -> 25)
- (Upcoming) Custom URL endpoints

# Development

B.U.F.F. is intended to be a small, month long side project for me to become fully immersed in full stack development, utilizing a Next.js front end with a separate FastAPI backend written in Python. I chose to utilize MongoDB to handle my database functions, as the ability to shard the database becomes increasingly important at scale.  

The site also features a full, working authentication system using FastAPI-users. OAuth2 features are currently being developed to complement the existing signup/login system.
